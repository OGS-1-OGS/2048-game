# 2048 Game

一个基于 JavaScript 实现的 2048 游戏，支持移动端和桌面端。

## 特性

- 响应式设计，完美适配移动端和桌面端
- 支持触摸滑动和键盘控制
- 平滑的动画效果
- 分数系统和最高分记录
- 自动保存最高分

## 如何使用

1. 在桌面端：
   - 使用方向键（↑↓←→）或 WASD 键控制方块移动
   - 点击"重新开始"按钮重置游戏

2. 在移动端：
   - 滑动屏幕控制方块移动
   - 点击"重新开始"按钮重置游戏

## 游戏规则

1. 相同数字的方块相撞时会合并成为它们的和
2. 每次移动后会在空白处随机出现一个新的方块（2或4）
3. 当无法移动时游戏结束

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 本地运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。
